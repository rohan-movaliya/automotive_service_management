This Automotive service Module facilitate the user to book their car appointment for service.
Book Service Appointment

Click on Book Appointment to open appointment Wizard

Now In wizard interface fill up all required fields and click on Book For create new appointment for service.

Now you can see new appointment record has been created.
After that in service state mechanic will working on appointed car

After that serviced state mechanic will complete appointed car service.

Now click on payment button
Wizard will be open and fill up all required fields and click on PAID


Now customer payment is done and its payment reference number will show at bottom.
